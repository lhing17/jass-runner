## pyproject.toml 的作用

`pyproject.toml` 是现代 Python 项目的**核心配置文件**，替代了传统的 `setup.py` 和 `setup.cfg`。

### 主要功能：
1. **项目元数据** - 名称、版本、作者、许可证等基本信息
2. **构建系统** - 指定构建工具（本项目用 setuptools + wheel）
3. **依赖管理** - 运行时依赖和可选的开发依赖
4. **包查找配置** - 指定包所在目录（本项目在 `src/` 下）

### 本项目配置要点：
- **包名**：`jass-runner`
- **Python版本**：>= 3.8
- **开发依赖**：`pytest`（测试）、`black`（代码格式化）、`flake8`（代码检查）
- **安装命令**：`pip install -e ".[dev]"` 安装全部依赖

### 为什么重要？
- 标准化配置，各种工具（pip、pytest、black）都能识别
- 一个文件统一管理，避免多个配置文件
- 支持现代 Python 打包和分发标准
